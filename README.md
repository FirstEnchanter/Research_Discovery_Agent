#  Insight Research Agent

A high-integrity discovery engine designed to identify external topics, compare them against the Agent content bank, and generate source-backed post ideas.

---

##  Objective
To surface non-duplicative, thought-leadership topics from **academic and institutional sources** that protect Autonomous Systems brand integrity while increasing reach.

---

##  Evaluation Lenses
Every topic discovered is analyzed through five professional "lenses" to ensure it aligns with the defined Agent voice:
1. **Alignment vs Misalignment**: Finding the gap between claim and reality.
2. **Structure as Reflection of Values**: Decoding what organizational structure reveals.
3. **Stewardship vs Extraction**: Evaluating long-term viability.
4. **Clarity as a Strategic Asset**: Differentiating complexity from confusion.
5. **Designing for Participation**: Analyzing inclusion and trust.

---

##  Logic & Constraints
- **Zero Internet Noise**: Only academic (`.edu`) and government (`.gov`) sources allowed.
- **The 45-Day Rule**: Automatically rejects any topic that mimics content approved in the last 45 days.
- **The 3-Question Gatekeeper**: A topic is only valid if its connection to a theme can be explained without stretching.

---

##  Project Structure
- **`main.py`**: The central research engine.
- **Google Sheets Integration**: 
    - Output: `Topic_Ideas`
    - Audit: `Rejected_Log`
